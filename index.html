Quick Safety Tips

  • Never run .exe/.msi files from unknown sources on PC.
  • Prefer official library/college sources or instructors for notes.
  • If a file asks for extra passwords, treat it as suspicious.
  • When in doubt, open PDFs in an isolated environment or mobile device with antivirus.
